    老师 这个path是什么 为什么要用它 麻烦讲解一下呢 刚学
    2019-05-28
     作者回复

    path 是 Node.js 里面的基础模块，可以通过 path 获取路径信息（ps: 可以稍微了解下 Node.js 的基础用法）。

    比如此处我们要把打包的输出目录设置到当前项目目录下的 dist 文件，那么 output 的 path 的路径应该是：当前项目路径+ dist，这个时候可以通过：path.join(__dirname, 'dist') 来组合得到。
