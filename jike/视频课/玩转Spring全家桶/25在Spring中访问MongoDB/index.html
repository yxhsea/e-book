    老师，我想问一下如果有mongo中存储了多个Document类型，是不是都会通过MoneyReadConverter 转换器转成Money类型？如果需要根据不同的Document转成不同的类型，除了写多个转换器在convert方法中用if判断之外还有没有其他更好的办法？谢谢
    我理解的逻辑：
    1. 老师定义的public class MoneyReadConverter implements Converter<Document, Money> 的意思是将Document 转成Money类型
    2. 在启动的时候spring会扫描Bean，将MoneyReadConverter 这个Bean注册到GenericConversionService.ConverterAdapter.converter (Converter<Object, Object>类型) 中
    3. 在执行query时，从mongo中取出来的“ { "money" : { "currency" : { "code" : "CNY", "numericCode" : 156, "decimalPlaces" : 2 }, "amount" : "20.00" } ”是Document类型的数据，所以触发了自定义的转换器MoneyReadConverter ，会将Docmuent转换成Money类型
    2019-02-28
     作者回复

    要转成什么类型是由我这个目标类的成员属性的类型决定的，如果这个位置我定义的是个Foo类型，那会找能把Document转成Foo的，如果定义的是个Bar类型，那就会找能把Document转成Bar的。所以你没必要在一个Converter里纠结，多定义几个Converter吧，分别处理不同的类型就好了。
